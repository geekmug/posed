Server Command-Line Options

  • --posed.save.cron=string (default: never) - Cron expression ("<second> <minute> <hour> <day> <month> <weekday>") for describing when to save the state.
  • --posed.save.filename=string (default: posed.yml) - Filename to use for saving the state of the Pose Engine.
  • --server.address=ip (default: wildcard) - What address the server should be bound.
  • --server.port=number (default: 8080) - What port the server should be bound.
